# Error Record Files

Use this directory for detailed error records that are too large to keep inline
in `../ERRORS_AND_LEARNINGS.md`.

File naming convention:

```text
EAL-YYYYMMDD-NNN-short-title.md
```

Create new files from `../templates/error-record.md`, then add a link in the
`Error Index` table in `../ERRORS_AND_LEARNINGS.md`.
